# tools 
- common utils plus makes your life easier.

## uuid 
param is a number which the length of return string value is
```javascript 1.8
import { uuid } from 'common-utils-plus'

console.log(uuid(36))
```

## shuffle
param is an array
```javascript 1.8
import { shuffle } from 'common-utils-plus'
let arr = [1, 2, 3, 4, 5]
console.log(shuffle(arr))
```
